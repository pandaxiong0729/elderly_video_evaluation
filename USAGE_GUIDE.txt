================================================================================
🔧 修改配置的方法
================================================================================

场景1：修改视频切片时长
------
方式1：通过命令行参数（推荐）
    python clips_tools.py --clip_duration 60

方式2：在 config.py 中修改
    CLIP_DURATION = 60  # 改为 60 秒

然后运行：
    python clips_tools.py
    # 自动使用新的 60 秒配置


场景2：修改低分阈值进行分析
------
方式1：通过命令行参数（推荐）
    python analyze_results.py --threshold 0.3

方式2：在 config.py 中修改
    BLEU1_LOW_SCORE_THRESHOLD = 0.3  # 改为 0.3

然后运行：
    python analyze_results.py
    # 低于 0.3 的视频会被标记为"低分"


场景3：修改路径（如果数据位置改变）
------
方式1：通过命令行参数
    python clips_tools.py --video_dir 新路径 --output_video_dir 新路径

方式2：在 config.py 中修改
    ORIGINAL_VIDEO_DIR = "新路径"
    CLIP_VIDEO_DIR = "新路径"

然后运行相关脚本，自动使用新路径


场景4：切换不同的评测指标组合
------
运行（无需修改任何代码）：
    python re_evaluate.py --profile bleu1_only      # 仅 BLEU-1
    python re_evaluate.py --profile bleu_family     # BLEU-1,2,3,4
    python re_evaluate.py --profile bleu_cer        # BLEU-1 + CER
    python re_evaluate.py --profile comprehensive   # 所有指标


================================================================================
📊 新的脚本使用方式
================================================================================

1. 视频切片工具 (clips_tools.py)
   -------
   python clips_tools.py
   
   特点：
   - 自动使用 config.py 中的路径和时长
   - 基于 SRT 时间戳精确对齐视频和字幕
   - 无需手动传递任何参数

   

2. 结果分析工具 (analyze_results.py)
   -------
   python analyze_results.py
   
   特点：
   - 自动使用 config.py 中的 CSV 路径和阈值
   - 分离低分和高分视频到两个 CSV 文件
   - 重新计算高分视频的平均 BLEU-1

   输出：
   ├── low_score_videos_below_0.5.csv
   └── high_score_videos_above_0.5.csv


3. 重新评测工具 (re_evaluate.py)
   -------
   python re_evaluate.py --profile bleu1_only
   
   可用的指标组合：
   - bleu1_only:      仅 BLEU-1（快速评估）
   - bleu_family:     BLEU-1, BLEU-2, BLEU-3, BLEU-4
   - bleu_cer:        BLEU-1 + CER（综合评估）
   - comprehensive:   所有指标（详细评估）

   输出：
   ├── re_evaluated_bleu1_only.csv
   ├── re_evaluated_bleu_family.csv
   ├── re_evaluated_bleu_cer.csv
   └── re_evaluated_comprehensive.csv

1. 所有 Python 脚本都会自动导入 config.py 中的配置

问题：ModuleNotFoundError: No module named 'config'
